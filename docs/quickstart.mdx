# Quickstart

## Prerequisites

- Cloudflare Workers project with `wrangler.jsonc`
- Neon Postgres database (for packages that need `@latimer-woods-tech/neon`)
- Node.js 20+

## Install packages

Install only the packages your app needs:

```bash
# Minimal: errors + logger + auth
npm install @latimer-woods-tech/errors @latimer-woods-tech/logger @latimer-woods-tech/auth

# Full stack
npm install @latimer-woods-tech/errors @latimer-woods-tech/monitoring @latimer-woods-tech/logger \
            @latimer-woods-tech/auth @latimer-woods-tech/neon @latimer-woods-tech/stripe \
            @latimer-woods-tech/analytics @latimer-woods-tech/llm @latimer-woods-tech/email
```

## Configure bindings

Add your secrets to `wrangler.jsonc` as bindings (never as plaintext `vars`):

```jsonc
// wrangler.jsonc
{
  "name": "my-app",
  "compatibility_date": "2025-01-01",
  "hyperdrive": [
    {
      "binding": "DB",
      "id": "<your-hyperdrive-id>"
    }
  ]
}
```

Store secrets via Wrangler:
```bash
wrangler secret put JWT_SECRET
wrangler secret put ANTHROPIC_API_KEY
wrangler secret put STRIPE_WEBHOOK_SECRET
```

## Wire up your Worker

```typescript
import { Hono } from 'hono';
import { jwtMiddleware, requireRole } from '@latimer-woods-tech/auth';
import { createLogger } from '@latimer-woods-tech/logger';

type Env = {
  DB: Hyperdrive;
  JWT_SECRET: string;
};

const app = new Hono<{ Bindings: Env }>();

app.use('*', (c, next) => {
  const logger = createLogger({ service: 'my-app' });
  c.set('logger', logger);
  return next();
});

app.use('/api/*', (c, next) => jwtMiddleware(c.env.JWT_SECRET)(c, next));
app.use('/api/admin/*', requireRole('admin'));

app.get('/api/me', (c) => {
  const user = c.get('user');
  return c.json({ sub: user.sub, role: user.role });
});

export default app;
```
