# Archive Policy

This folder stores historical planning and session artifacts that should not be used as active execution sources.

## Rules

1. Canonical execution sources are:
- `CLAUDE.md`
- `WORLD_CLASS_IMPLEMENTATION_DASHBOARD.md`
- `docs/operations/WORLD_CLASS_360_TASK_DASHBOARD.md`
- `docs/service-registry.yml`

2. Any file moved here is evidence-only and must not be treated as a current backlog.

3. If a historical file remains at repository root, add the banner:

`> HISTORICAL DOCUMENT: superseded by WORLD_CLASS_360_TASK_DASHBOARD.md and WORLD_CLASS_IMPLEMENTATION_DASHBOARD.md`

4. Do not delete historical artifacts automatically; preserve for auditability unless explicitly approved.
