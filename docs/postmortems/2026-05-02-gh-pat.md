# 2026-05-02: GH_PAT expired, supervisor fatal error

**Severity:** P3  **Duration:** ~10 minutes  **Impact:** First supervisor run failed, Pushover alert sent

## What happened
`GH_PAT` org secret held an expired personal access token. Supervisor got `401 Bad Credentials` reading template files. Pushover alert fired correctly.

## Root cause
Long-lived PATs expire. Automated workflows should not depend on manually-rotated personal tokens.

## What we changed
- Supervisor uses `github.token` for factory repo reads (PR #157)
- Added expired credential pattern to `security.md`
- TODO: wire Factory GitHub App token for cross-repo writes
