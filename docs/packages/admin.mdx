# @latimer-woods-tech/admin

Drop-in Hono admin router with user management, event inspection, and health check endpoints. Depends on `@latimer-woods-tech/auth` and `@latimer-woods-tech/analytics`.

## Installation

```bash
npm install @latimer-woods-tech/admin
```

## API

### `createAdminRouter(opts)`

Returns a Hono sub-application to mount at an admin path.

```typescript
function createAdminRouter(opts: {
  db: Db;
  appId: string;
  requireAdminRole?: boolean; // default true, enforces 'admin' role via requireRole()
}): Hono;
```

## Routes

| Method | Path | Description |
|---|---|---|
| `GET` | `/` | Dashboard summary: `totalUsers`, `activeUsers`, `recentEvents` |
| `GET` | `/users` | Paginated user list. Query: `?limit=20&offset=0` |
| `GET` | `/users/:id` | User detail with subscriptions |
| `POST` | `/users/:id/suspend` | Suspends a user (sets `status = 'suspended'`) |
| `GET` | `/events` | Recent `factory_events` for the app. Query: `?limit=50` |
| `GET` | `/health` | DB connectivity ping. Returns `{ status: 'ok' | 'degraded' }` |

All routes return `401` without a valid JWT and `403` without the `admin` role (when `requireAdminRole` is true).

## Example

```typescript
import { createAdminRouter } from '@latimer-woods-tech/admin';
import { createDb } from '@latimer-woods-tech/neon';
import { jwtMiddleware } from '@latimer-woods-tech/auth';
import { Hono } from 'hono';

const app = new Hono<{ Bindings: Env }>();

// Global JWT guard
app.use('*', (c, next) => jwtMiddleware(c.env.JWT_SECRET)(c, next));

// Mount admin router
app.route('/admin', createAdminRouter({
  db: createDb(env.DB),
  appId: 'focusbro',
}));
```
