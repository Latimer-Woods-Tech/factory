## Appendix A: Monthly Close Checklist

**By 5:00 PM UTC on last business day of month:**

### Financial Completion (Finance Lead)
- [ ] All daily revenue transactions in database
- [ ] All refunds/chargebacks recorded
- [ ] Creator earnings calculated and allocated
- [ ] All payouts recorded (completed + pending)
- [ ] Reconciliation balanced (variance <$100)
- [ ] Foreign exchange (if applicable) settled
- [ ] Sales tax / GST accounted for (if applicable)

### Operational Verification (Ops)
- [ ] Payout pipeline clear (no stuck batches)
- [ ] DLQ payouts addressed (manual retries or escalated)
- [ ] Creator account issues resolved
- [ ] Stripe account reconciled against bank statement
- [ ] Failed transactions logged and triaged

### Process Completion
- [ ] P&L summary generated
- [ ] Variance analysis completed
- [ ] Risk exceptions documented
- [ ] Audit export prepared
- [ ] Executive summary drafted

### Sign-Off
- [ ] Finance lead approval
- [ ] CFO approval
- [ ] Auditor notification (if scheduled)
- [ ] Internal distribution (team, executives)

## Appendix B: Monthly Close Query Reference

```sql
-- Query 1: Total Revenue (Subscriptions + Unlocks)
SELECT
  SUM(amount_cents) / 100.0 as total_revenue_usd
FROM factory_events
WHERE event_name IN ('subscription_payment_succeeded', 'unlock_payment_succeeded')
  AND status = 'success'
  AND DATE_TRUNC('month', event_timestamp) = DATE_TRUNC('month', CURRENT_DATE);

-- Query 2: Creator Earnings Recorded
SELECT
  SUM(amount_cents) / 100.0 as total_earnings_usd
FROM factory_events
WHERE event_name = 'creator_earnings_recorded'
  AND status = 'success'
  AND DATE_TRUNC('month', event_timestamp) = DATE_TRUNC('month', CURRENT_DATE);

-- Query 3: Payouts Completed
SELECT
  SUM(amount_cents) / 100.0 as total_payouts_usd
FROM factory_events
WHERE event_name = 'payout_completed'
  AND status = 'success'
  AND DATE_TRUNC('month', event_timestamp) = DATE_TRUNC('month', CURRENT_DATE);

-- Query 4: Refunds & Chargebacks
SELECT
  SUM(amount_cents) / 100.0 as refund_total_usd
FROM factory_events
WHERE event_name IN ('subscription_cancelled', 'refund_issued')
  AND DATE_TRUNC('month', event_timestamp) = DATE_TRUNC('month', CURRENT_DATE);
```
